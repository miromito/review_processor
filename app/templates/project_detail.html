Графики

Фильтр по ключевому слову:
Сводка тональности
Подсказка

Кольцевая диаграмма показывает долю отзывов в разрезе тональности: позитив (зелёный), негатив (красный), нейтраль (серый), прочее (оранжевый).

Ориентир: преобладание зелёного сегмента указывает на благоприятный общий фон оценок; существенная доля красного свидетельствует о выраженной критике. Сводка строится по полному набору отзывов проекта и не привязана к отдельным датам.

Ключевые слова
Подсказка

До 14 самых частых ключевых слов (1–2 слова в разметке). Размер на «карте» отражает частоту в текущем срезе (дата, тема, фильтры из файла).

Щёлкните по слову — оно станет фильтром для всех графиков; повторный щелчок по выбранному снимает фильтр. Для старых проектов без ключевых слов перезапустите анализ.

Нет данных по ключевым словам для текущего среза. Если проект анализировался до появления этой функции, запустите анализ снова.
Динамика тональности и доля позитивных
Подсказка

Левая ось (числа): количество отзывов по каждой тональности за календарный день. Правая ось (проценты, без дробной части): доля позитивных отзывов за день; сплошная линия — значение за день, пунктир — скользящее среднее за семь дней для оценки тенденции.

При малых суточных объёмах (1–3 отзыва) доля позитивов может сильно колебаться; ориентир — пунктир и сопоставление с зелёной/красной кривой слева. Щелчок по маркеру на линии тональности открывает перечень отзывов за дату (с учётом фильтров). Для графика требуется колонка даты в проекте.

Темы и тональность
Подсказка

Каждая полоса — одна тема. Сегменты — тональности (позитив, нейтраль, негатив) среди отзывов с этой темой. У отзыва одна тема — одна полоса на отзыв.

Интерпретация: сочетание большой длины полосы с заметной долей негативного сегмента указывает на важную тему с критичными оценками, даже при высоком общем числе упоминаний.

«Боль»: негатив по теме
Подсказка

Диаграмма сопоставляет темы по количеству негативных отзывов; длина красного столбца и всплывающая подсказка отражают объём и удельную долю критичных оценок по теме.

Ориентир: тема с умеренным количеством упоминаний, но высокой долей негатива, может требовать приоритетного внимания по сравнению с темой с большим числом, но преимущественно нейтральных откликов.

Дата, тональность и тема
Подсказка

Каждый пузырёк — шаг по времени и тональность. Размер — число отзывов, цвет — доминирующая тема в группе. Клик — список отзывов. Нужна колонка даты в файле.

Сетка по горизонтали: по дням — календарные даты; по неделе, месяцу, кварталу или году — одна колонка на период, тональности визуально разделяются по вертикали. При слишком плотной сетке по дням рекомендуется укрупнить шаг (неделя, месяц).

Краткий бизнес-инсайт
Подсказка

Краткое заключение формируется автоматически после завершения анализа. Модель использует сводные показатели и фрагмент выборки (первые 50 записей с префиксом K) и формулирует приоритеты для внимания.

Инсайт относится к проекту в целом. Параметры отображения графиков (дата, тема, поля из файла) на текст заключения не влияют.

Фильтры
Подсказка
Таблица содержит обработанные отзывы (записи с префиксом K). Тональность — итог по тексту. Тема — одна метка на отзыв; в фильтре — точное совпадение. По 10 строк, страницы внизу и вверху.
